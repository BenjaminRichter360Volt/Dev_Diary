# Speichern für Später:
- Azure Portal Öffnen
- Microsoft Entra ID => Gruppen => Neue Gruppe
	- Gruppentyp: Sicherheit
	- Gruppenname: AZ-[TEAM]-[ROLE] | AZ-DEV-Contributor
	- Beschreibung: z.B. Entwickler mit Contributor-Rechten
	- Mitgliedschaftstyp: Zugewiesen
- Gruppe AZ-DEV-Contributor öffnen
	- Mitglieder => Mitglied hinzufügen
	- Entwickler auswählen
	- Hinzufügen
- Azure RBAC Rolle zuweisen => Scope wählen
	- Option auswahl:
		- Ganze Subscription: OK für kleine Teams
		- Resource Group: Best Practice
		- Managment Group: Nur bei großen Strukturen
	- Rolle auswählen => Contributor
	- Mitglieder auswählen
		- Assign access to: User, group or service principal
		- Select members: AZ-DEV-Contributor
	- Review + assign
- Ergebnis testen: Hinweis RBAC kann 5-10 Minuten brauchen
